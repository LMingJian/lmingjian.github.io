title: 如何将两个路由器设置成同一网段

# 需求

现有 AB 两个主副路由器，其中主路由器 A 连接外网，通过一条网线连接副路由器 B。

此时，用户需要所有连接主副路由器 AB 的设备网段都一样，在一个局域网中。

![](_images/drawingbed/img/Pasted%20image%2020251208141739.png)

# 实现

**第一步，登录副路由器 B 的管理后台，关闭 DHCP 配置**。

> 这一步是让副路由器停止 IP 分配，让所有设备的 IP 分配通过主路由器进行。

![](_images/drawingbed/img/Pasted%20image%2020251208145144.png)

**第二步，找到主路由器 A 的 LAN 配置，记录其的 IP 地址**。

**第三步，登录副路由器 B 的管理后台，找到 LAN 配置，将其修改为与主路由器 A 同一网段，但不同地址的 IP。比如，主路由器 A 的 LAN IP 是 192.168.0.1，那么副路由器 B 的 LAN 口就可以设置为 192.168.0.2**。

> 这一步是为了让副路由器与主路由器通过 LAN 口相连时，避免出现 IP 冲突，让副路由器变成一台交换机。

![](_images/drawingbed/img/Pasted%20image%2020251208145724.png)

**最后，将副路由器 B 的 LAN 口与主路由器 A 的 LAN 口进行相连，此时，所有接入 AB 的设备，其网段都是一样的**。

![](_images/drawingbed/img/Pasted%20image%2020251208150658.png)
