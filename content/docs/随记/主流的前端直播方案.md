# 直播协议

- **RTMP**：底层基于 TCP，在浏览器端依赖 Flash。
- **HTTP-FLV**：基于 HTTP 流式 IO 传输 FLV，依赖浏览器支持播放 FLV。
- **WebSocket-FLV**：基于 WebSocket 传输 FLV，依赖浏览器支持播放 FLV。
- **HLS**：基于 HTTP 的分片传输流媒体协议，HTML5 可以直接打开播放，M3U，M3U8 就是这种协议的实现。
- **RTP**：底层基于 UDP，延迟低至 1 秒，但浏览器不支持播放。

# 主流协议间的性能

|传输协议|播放器|延迟|
|---|---|---|
|RTMP|Flash|1s|
|HTTP-FLV|HTML Video|1s|
|HLS|HTML Video|20s|
