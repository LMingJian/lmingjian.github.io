# Type-C 接口异常导致的

请严格按照如下操作步骤执行：

- 确保机器处于关机状态；  
- 移除电源适配器连接；  
- 同时按下 Fn+S+V 几秒，按开机键确认无法开机；
- 连接电源适配器开机；
- 测试 Type-C口 功能是否恢复正常。

![](_images/drawingbed/img/Pasted%20image%2020260402100739.png)

# 扩展坞投屏异常

使用充电的 C 口连接拓展坞，存在概率另外一个 C 口的功率不足。
