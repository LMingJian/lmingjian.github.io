# [Zabbix](https://www.zabbix.com)

‌**Zabbix 是一个企业级的开源分布式监控系统**‌，能够实时监控服务器、网络设备、应用程序、数据库、云服务等多种 IT 基础设施的性能和健康状态。

它通过灵活的数据采集方式（如 Agent 代理、SNMP、IPMI、JMX、VMware 监控等）获取关键指标，比如 CPU 使用率、内存占用、磁盘空间、网络流量、服务状态等，并将这些数据可视化为图表、仪表盘和拓扑图，帮助运维人员全面掌握系统运行状况 。
