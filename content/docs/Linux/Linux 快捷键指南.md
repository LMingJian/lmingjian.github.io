# 基础操作快捷键

## 光标移动

| 快捷键     |  功能   |
| --- | --- |
|  Ctrl + A   |  快速定位到行首   |
|  Ctrl + E   |  快速定位到行尾   |
|  Ctrl + F   |  向行尾移动一个字符（等价于左方向键）   |
|  Ctrl + B   |  向行首移动一个字符（等价于右方向键）   |
|  Alt + F   |  向行尾移动一个单词   |
|  Alt + B   |  向行首移动一个单词   |

## 文本编辑

| 快捷键     |  功能   |
| --- | --- |
|  Ctrl + U   |  删除光标到行首的内容    |
|  Ctrl + K   |  删除光标到行尾的内容   |
|  Ctrl + W   |  删除光标前的一个单词   |
|  Ctrl + D   |  删除光标后的字符   |
|  Ctrl + Y   |  粘贴之前删除的内容   |
|  Ctrl + L   |  清屏（保留历史记录）   |

## 命令管理

| 快捷键     |  功能   |
| --- | --- |
|  Ctrl + P   |  上一条历史命令（等价于上方向键）   |
|  Ctrl + N   |  下一条历史命令（等价于下方向键）   |
|  Ctrl + R   |  搜索历史命令   |
|  !!   |  直接执行上一条命令   |
|  ESC + .   |  粘贴上一条命令的最后一部分   |

# 其他快捷键

## 进程管理

| 快捷键     |  功能   |
| --- | --- |
| Tab   |  自动补全命令/路径/文件名 |
| Ctrl + C   |  中断当前命令 |
| Ctrl + D   |  退出当前 Shell（相当于exit） |

## Vi 或 Vim 快捷键

| 快捷键     |  功能   |
| --- | --- |
| i   |  进入插入模式 |  
| ESC   |  返回普通模式 |  
| :   |  进入命令模式 |  
| :wq |  保存并退出 | 
| :wq |  保存并强制退出 |  
| :q |  不保存并退出 |
| :q! |  不保存并强制退出 |
| / | 进入搜索模式 |
| h/j/k/l |  左下上右移动 |  
| 0 |  行首 |  
| $ |  行尾 |  
| gg |  文件首行 |  
| G |  文件尾行 |  
| dd |  删除整行 |  
| yy |  复制当前行 |  
| p |  粘贴 |
