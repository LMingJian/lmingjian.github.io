# 需求

在编写 Shell 脚本时，有时候需要检查前一命令是否执行成功。

此时可以在 if 条件中使用 `$?` 来检查前一命令的结束状态。

# 示例

```bash
if [ $? -eq 0 ]; then  
    echo "执行成功"  
else  
    echo "执行失败"  
fi
```

- 如果结束状态等于 0，说明前一个命令执行成功。
- 如果结束状态不是 0，说明前一个命令执行失败。
