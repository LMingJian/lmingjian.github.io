# 版本分类

|  版本   | 内容    |
| --- | --- |
|  CentOS-7.0-x86_64-**DVD**-1503-01.iso   | 标准安装版    |
| CentOS-7.0-x86_64-**NetInstall**-1503-01.iso   | 网络安装版     |
| CentOS-7.0-x86_64-**Everything**-1503-01.iso   | 集成所有软件的完整安装版    |
| CentOS-7.0-x86_64-**GnomeLive**-1503-01.iso   | GNOME 桌面版    |
| CentOS-7.0-x86_64-**KdeLive**-1503-01.iso   | KDE 桌面版    |
| CentOS-7.0-x86_64-**livecd**-1503-01.iso   | 光盘版，类似于 Windows PE    |
| CentOS-7.0-x86_64-**minimal**-1503-01.iso   | 自带软件最少的最小安装版   |
