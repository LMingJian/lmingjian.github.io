# 安装验证

在 Cmd 控制台或 PowerShell 控制台输入以下命令，检查输出：

```python
python --version
pip list
```

![](_images/drawingbed/img/Pasted%20image%2020250903153451.png)

如果执行失败，出现问题，请查看系统变量是否正常，可查阅以下文章。
