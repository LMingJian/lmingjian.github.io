# 需求

在使用 [scrcpy](https://github.com/Genymobile/scrcpy) 这一安卓投屏工具时，如果要时刻使用 USB 线进行连接那十分麻烦。此时，如果通过 adb 的无线调试功能，通过网络连接手机，便可以实现无线远程投屏控制手机。

# 启用生产模式

华为设备默认不会启动无线调试，因此，需要用户进入生产模式后才能使用无线调试。

用户可以通过以下方式来打开工程菜单，将调试模式变更为生产模式。

- 在**拨号界面**中输入代码 `*#*#2846579#*#*`，即可进入工程菜单。

在工程菜单，找到**后台设置-USB端口设置**，即可将调试模式变更为生产模式。

# 启用无线调试

在开启设备能力后，我们还需要通过 adb 开启设备的无线调试端口，这样才能远程连接设备。

按以下步骤，通过 adb 开启无线端口 5555：

1. 使用 `USB` 连接手机。
2. 执行 `adb tcpip 5555` ，开启无线端口。
3. 执行 `adb connect 设备IP:5555` ，在同一网络环境下连接设备。
4. 连接成功后，即断开 `USB`，此时设备已经通过无线连接到电脑了。
5. 执行 `adb devices` ，此时可以查看到连接手机状态以及基本信息。
