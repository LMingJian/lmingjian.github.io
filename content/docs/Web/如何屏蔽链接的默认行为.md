# 问题

在 HTML 中，`<a>` 标签通常用于创建链接，默认情况下，用户点击`<a>` 标签回执行其默认行为，导航到一个新的页面，但大多数时候，我们并不希望通过这个默认行为直接跳转，希望在点击后，链接会保持在当前页面，同时可以通过 javascript 添加其他功能，如触发弹窗、开启模态框等。

# 解决

```html
<a href="javascript:void(0);">something</a>
```
