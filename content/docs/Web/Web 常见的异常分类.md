# 常见的前端异常

- JS 代码执行异常
- Promise 异常
- 静态资源加载异常
- console.error 异常
- 跨域异常

# 常见的接口异常

接口异常属于后端的异常，接口异常会直接导致前端页面错误，因此这类异常是我们判断线上问题根源的重要依据。接口异常可以根据响应结果分类：

- 未响应/超时响应异常
- 4xx 请求异常
- 5xx 服务器异常
- 权限不足异常
